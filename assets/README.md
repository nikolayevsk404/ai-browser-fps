# Assets

Reserved folder for source assets that will replace the current procedural placeholders.

```txt
assets/
  3d/
    maps/          Future map models and source exports.
    characters/
      ct/          Future CT character models.
      tr/          Future TR character models.
    weapons/       Future weapon models.
```

Keep generated runtime assets in the application folders only after an import pipeline exists. Until then, this folder documents the intended source-asset layout.
