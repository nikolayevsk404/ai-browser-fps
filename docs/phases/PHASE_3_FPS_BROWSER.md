# Fase 3 - FPS Browser

## Entregue

- Cena Three.js full-screen.
- Câmera em primeira pessoa.
- Movimento WASD.
- Mouse look com Pointer Lock.
- Colisão simples no client.
- Viewmodel de arma.
- Raycast shooting.
- Bots low poly renderizados a partir do snapshot do servidor.
- HUD com placar, tempo, vivos, HP, arma e respawn.

## Decisão Atual

O servidor continua autoritativo para tiros, dano, morte, respawn, placar e fim da partida. O client renderiza e envia inputs.
